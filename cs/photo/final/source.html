Overview/Presentation Video:

Source Video

The video I used as a source for all of my results is that of Gabe the dog. This video was uploaded roughly three years ago with the intent for the internet to remix it, spawning countless music videos starring the furry diva.

Initial Tests

  • I began testing with a C-major Scale to make sure that my code worked for the simplest case.

  • I then added a second track transposed by a 5th to make sure that the code worked with more than one track .

From here, the sky's the limit. I took a number of fan-made MIDIs of songs from various game soundtracks off of vgmusic.com. And used them as the blueprint for the following videos.

Track Selection Differences

The following are three videos using the same midi file as a blueprint, but selecting different tracks from the midi file to be remixed.

  • Using the two main melody tracks

  • Using the two melody tracks and a single bass track. Notice that the bass line causes the remix to feel much too full as it overlaps with all the other tracks.

  • Eschewing the bass track for two harmony tracks. This sounds much better.
