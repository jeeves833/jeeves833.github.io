Final Project: Mesh Simplification

Lucas Karahadian

In this project, I will be implementing mesh simplification using a quadric error metric. This will involve ranking the existing edges such that higher ranked edges contribute more to the overall shape of the mesh. Then I will collapse the edges in this order until reaching a user-specified number of edges in the mesh. This will be built on top of the existing MeshEdit program from assignment 2.

Background

It comes as no surprise to anyone doing high-level 3D modeling that meshes can grow to an enormous size. The largest of the meshes we have dealt with reach up to 250,000 faces. In addition to taking up more memory, these large meshes can drammatically slow down the process of ray-tracing in physics based rendering programs. The mesh simplification process collapses edges to reduce the number of faces in a way such that the mesh looks overall similar and is (in results) akin to downsampling textures. I will be basing my code on this paper. Additionally, as a practical application for this simplification, I propose that these simplified meshes can substitute far away meshes or small meshes for speeding up ray-tracing.

Resources

As stated above, I will be building this program on top of the previously built MeshEdit assignment here. The level of simplification will be controlled in a way similar to how we controlled the depth of the sensor plane in the Lens Simulation assignment. The "I" key will be used to simplify the mesh either by a discrete number of faces or to a percentage of faces. The "J" and "K" keys will be used to adjust the delta between simplification levels.

Goals & Deliverables

What I will deliver is an updated MeshEdit program that allows the user to simplify their meshes on the fly.

My primary goal is to demonstrate that mesh simplification via the quadric error metric can be used to reduce the size of a mesh while maintaining its viability for use in scenes that appear small from the camera's perspective. The primary deliverables will be photos of a collection of different meshes showing increasing levels of simplification. For my presentation I will also demonstrate the changes to the MeshEdit UI I made in regards to interactive simplification.

If I have time at the end, I may look at other metrics for mesh simplification and try to draw a comparison between them.
